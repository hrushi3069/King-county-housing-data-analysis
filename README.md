# King County Housing Data Analysis

## Overview

Exploratory data analysis of the King County Housing Dataset using Python, data mining techniques, statistical analysis, and data visualisation methods to identify key factors affecting house prices.

This project analyses housing features such as living area, bedrooms, bathrooms, location, grade, waterfront views, and other property characteristics to understand their relationship with house prices.

## Objectives

- Perform exploratory data analysis on the housing dataset
- Analyse statistical patterns and distributions
- Visualise relationships between housing features and prices
- Identify important factors influencing house value
- Prepare data for future machine learning modelling

## Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed

- Data cleaning and preprocessing
- Missing value analysis
- Duplicate record removal
- Date conversion and feature extraction
- Descriptive statistical analysis
- House price distribution analysis
- Correlation matrix analysis
- Geospatial price analysis
- Feature relationship visualisation

## Dataset

King County Housing Dataset containing property information including:

- Price
- Bedrooms
- Bathrooms
- Living area
- Lot size
- Floors
- Waterfront
- Location
- House grade
- Year built
